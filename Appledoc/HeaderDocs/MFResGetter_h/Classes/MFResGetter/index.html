MFResGetter

Superclass:
NSObject
Declared In:
MFResGetter.h

Introduction

Provides common resources access system methods replacements.

Discussion

MFResGetter implements an access to ressources in a bit safer and more versatile way than system equivalent methods. Method syntax is following the equivalent system methods syntax.

defaultMediaGetter

+(instancetype)defaultMediaGetter;
Return Value

Discussion

Returns the default MFResGetter. Several MFResGetter with different behaviors and base directory can be used in one project, but one will often be enough.

Properties

baseDirectoryPath

The path of the base directory for this getter. If NULL, always search the root directory. baseDirectoryPath is NULL by default

behavior

Behavior Options Set MFResGetterUseDefault bit to return default image Set MFResGetterSearchRoot bit to search at root if resource not found in base directory

bundle
debugMode

Debug options Set MFResGetterLog bit to activate logging Set MFResGetterLogFile to activate logging to file ( MFResGetterLog bit must be set ) Set MFResGetterBreaks to braks if a resource is missing ( MFResGetterLog bit must be set )

defaultImage

May be NULL. Default image is initialized at first access.

defaultImageName

name of the default image.

WARNING:

Default image must be a png located at bundle root
