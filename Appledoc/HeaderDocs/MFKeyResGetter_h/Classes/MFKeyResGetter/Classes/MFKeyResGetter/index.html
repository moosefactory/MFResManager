MFKeyResGetter

Superclass:
MFResGetter
Declared In:
MFKeyResGetter

Introduction

Provides access to (localized) resources through a table of content.

Discussion

MFKeyResGetter if a MFResGetter subclass and so provides all of its features. It adds resource loading via a table of content stored as a PLIST in the baseDirectory or root. It also adds support for localized resources
